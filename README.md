### Como rodar o projeto:
```
# Clone o projeto
git clone https://github.com/FilipeDiasLima/Ignite-Shop.git

# Abra o projeto
cd ./Ignite-Shop

# Instale as dependências
yarn ou npm install

# Execute
yarn dev
ou
yarn build
yarn start
```
